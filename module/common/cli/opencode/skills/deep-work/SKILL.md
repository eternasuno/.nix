# Deep Work

Complex task workflow for multi-phase sessions.

## Structure

Phase 0: Clarify    → `references/clarify.md`
Phase 1: Design     → `references/design.md`
Phase 2: Implement  → `references/implement.md`
Phase 2.5: Simplify → `references/simplify.md`
Phase 3: Verify     → `references/verify.md`
Phase 4: Review     → `references/review.md`
Phase 5: Handoff    → `references/handoff.md`

Track progress in `.slim/deepwork/<slug>.md`. Read each phase file when ready to begin that phase.

## Transition

Each phase completes when its defined output is produced. Move to the next phase only when the current output is validated.
